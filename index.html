Nossas Animações

Fade In Effect

Um efeito suave de aparecimento.

.animate-fade-in { animation: fadeIn 0.6s ease-out; }

Slide In Left

Elemento desliza da esquerda.

.animate-slide-in-left { animation: slideInLeft 0.7s ease-out; }

Pulse Effect

Um efeito de pulsação sutil.

.animate-pulse { animation: pulse 2s infinite; }

Rotate In

Elemento aparece girando.

.animate-rotate-in { animation: rotateIn 0.8s ease-out; }

Bounce Effect

Um efeito de salto divertido.

.animate-bounce { animation: bounce 1s ease-in-out; }

Flip In X

Elemento vira no eixo X.

.animate-flip-in-x { animation: flipInX 1s ease-out; }
